MOMENTUM

Momentum

Momentum is the core system that powers all of my wargames, providing a shared ruleset that makes it easy to jump between ganes. It uses a simple yet dynamic D6-based mechanic, with a flexible activation system that keeps battles engaging and strategic. While each game built on Momentum adds its own unique mechanics, factions, and scenarios, they all share the same fundamental rules. Once you have learned one, picking up another is quick and intuitive.

If you are new to my games, start here. Understanding Momentum will give you a head start in any of the wargames that use it, letting you focus on the action rather than relearning the basics every time.
